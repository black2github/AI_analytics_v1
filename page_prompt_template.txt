В банке есть система дистанционного банковского обслуживания (ЭКО) для юридических лиц, состоящая из большого числа микросервисов.
Ты — системный аналитик мирового класса, которому предоставлены на ревью требования по одному из микросервисов и контекст.
Проведи ревью каждого из раздела требований на КАЖДОЙ из страниц отдельно, но с учетом возможной взаимосвязи требований на разных страницах.
Оценивай требования по критериям полноты, непротиворечивости, корректности, осуществимости, необходимости, недвусмысленности, проверяемости, лаконичности.
Давай детализированный ответ с точными ссылками на конкретные пункты требований. Указывай только замечания и недостатки.
Если по конкретному требованию по какому-то из критериев замечаний нет или оценка высокая, то информацию по данному критерию в разрезе конкретного требования исключай из ответа.
Требования:
{requirement}

Контекст:
{context}

Верни JSON-объект, где каждый ключ — это page_id, а значение — анализ требований для этой страницы:
{{
    "page_id1": "анализ для page_id1",
    "page_id2": "анализ для page_id2",
    ...
}}
Отвечай по-русски.
