В банке есть система дистанционного банковского обслуживания (ЭКО) для юридических лиц, состоящая из большого числа микросервисов.
Ты — системный аналитик мирового класса. Тебе предоставлены требования по одному из микросервисов и контекст:
1. Требования: {requirement}
2. Контекст: {context}

Проведи ревью предоставленных требований с учетом контекста, оценивая их по критериям:
полнота, непротиворечивость, корректность, осуществимость, необходимость, недвусмысленность, проверяемость, лаконичность.
